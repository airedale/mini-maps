Minimum code to place a minimap on the page

Include in the <head> section, reference to:

jQuery library: <script src="http://code.jquery.com/jquery-1.7.min.js"></script>

Mini-map javascript functions: <script src="/js/mini_maps_min.js"></script>

CSS class:
    <style type="text/css">
    
        .expandable { 
            width: 300px; /* sets the map width */        
        }
  
    </style>
In the page <body>:

Include reference to the file named "smallmap_min.htm" and if an animated GIF image named "wait.gif" exists in the same location, it will be displayed before the map is rendered.

    
<div id="SomeIdName" class="expandable">

    <iframe src="smallmap_min.htm?lat=-31.9&lon=115.8" width="100%" height="100%" frameborder="0" scrolling="no"></iframe></div>

<script type="text/javascript">resize16x9("SomeIdName");</script>

Required files: "smallmap_min.htm", "wait.gif"(optional)

Parameters can be entered in any order:

  • 'latitude' or 'lat' = decimal (required)
  • 'longitude' or 'long' or 'lon' = decimal (required)
  • 'mzoom' or 'mapzoom' = integer [1-20]
  • 'caption' = string
  • 'mstreetview' or 'streetview' = boolean [true/false]
  • 'mzoomcontrol' or 'mapzoomcontrol' = boolean [true/false]
  • 'mtypecontrol' or 'maptypecontrol' = boolean [true/false]
  • 'mtypeid' or 'maptypeid' = integer [0=ROADMAP(default); 1=SATELLITE; 2=HYBRID; 3=TERRAIN]
  • 'marker' = boolean [true/false]
OK for multiple maps per page
